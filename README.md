# Note Uygulamam

Kendi geliştirdiğim note uygulamam

## Açıklama

Bu proje, kullanıcıların notlarını yönetebilecekleri basit fakat kullanışlı bir not uygulamasıdır.

## Teknolojiler

- HTML
- CSS
- JavaScript

## Özellikler

- Not ekleme
- Not silme
- Not düzenleme
- Not listeleme

## Kurulum

1. Repository'yi klonlayın:
```bash
git clone https://github.com/E-rdem19/Note_Uygulamam.git
```

2. Proje klasörüne gidin:
```bash
cd Note_Uygulamam
```

3. Uygulamayı tarayıcıda açın.

## Kullanım

Uygulamayı başlattıktan sonra:
1. Not başlığı ve içeriğini girin
2. "Ekle" butonuna tıklayın
3. Notlarınızı görüntüleyin ve yönetin
